# TDLC Documentos · GitHub Pages

Repositorio estático que recibe las copias de los PDF publicados por `informe-tdlc2`.

## Configuración

En GitHub:

```text
Settings → Pages
Source: Deploy from a branch
Branch: main
Folder: /docs
```

El programa creará automáticamente:

```text
docs/.nojekyll
docs/pdf/YYYY-MM-DD/<identificador>.pdf
```

Los documentos no se listan en la portada.
